## 🎯 恶搞模式

### 🖥️ 强制全屏（默认）

触发后弹出占据整个屏幕的恶搞页面：
- 覆盖所有 UI 元素（`z-index: 2147483647`）
- **无法关闭** — 没有关闭按钮，屏蔽所有键盘/鼠标事件
- 自动尝试浏览器全屏 API
- 随机显示 5 种不同的恶搞文案
- 每 3 秒飘出浮动 emoji 粒子动画
- 停止方法：在 EchoMusic 插件管理中禁用此插件

### 💀 假死（无响应）

触发后阻塞主线程使窗口完全无响应：
- 使用 `Atomics.wait` 或 `while(true)` 死循环
- 只能通过任务管理器结束 EchoMusic 进程

## 🚀 使用方式

1. 在 EchoMusic 中加载此插件
2. 插件激活后自动开始倒计时
3. 等待随机延时到达后自动触发恶搞
4. 或在设置面板点击「立即执行」手动触发

### 命令

- `prank-system.test` — 立即执行当前恶搞模式
- `prank-system.cancel` — 取消倒计时
