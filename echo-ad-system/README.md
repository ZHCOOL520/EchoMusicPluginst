# EchoMusic 广告系统插件 v1.0.0

开屏广告 + 播放中插播广告 + 启动音效，支持 Bing 每日壁纸随机背景、自定义图片与弹窗时间。

**作者: [ZHCOOL520](https://github.com/ZHCOOL520)**

## ✨ 功能特性

### 🔊 启动音效
- 软件启动时自动播放音频（如「嗨喽酷狗」）
- 支持插件目录内相对路径、file:// 本地路径、https:// 在线地址
- 音量可调节（0~100%）
- 可一键开关，可自定义任意音频文件
- 内置测试播放按钮

### 🚀 开屏广告
- 应用启动时全屏弹出，带倒计时进度条
- 展示时长可配置（3~30 秒）
- 可跳过时间可配置
- 从广告池随机选取，多条广告轮换

### 🎵 播放中插播广告
- 音乐播放随机时长后自动暂停，弹出广告
- 触发间隔可配置最短/最长
- 广告展示时长可配置（3~60 秒）
- 倒计时结束后自动恢复播放
- 仅在播放状态触发，暂停时不打扰

### 🖼️ Bing 每日壁纸
- 今日壁纸 / 近 8 天随机 / 自定义列表 / 混合模式 / 纯色背景

### 🎨 用户自定义
- 每条广告独立配置：标题、副标题、跳转链接、背景色、文字色、强调色
- 图片来源 5 种模式可选
- 自定义图片列表支持多张
- 设置持久化，重启不丢失

## 📦 安装

### 在线安装
1. EchoMusic → 插件管理 → 添加插件源
2. 源地址：`https://github.com/hoowhoami/EchoMusicPlugins`
3. 在线插件 → 安装「广告系统」

### 手动安装
将 `echo-ad-system` 文件夹复制到 EchoMusic 插件目录，刷新启用。

## ⚙️ 配置说明

| 设置项 | 默认值 | 说明 |
|--------|--------|------|
| 全局开关 | ✅ | 总开关 |
| 启动音效 | ✅ | 启动时播放音频 |
| 音频文件 | audio/hello-kugou.mp3 | 支持相对/绝对/URL |
| 音量 | 80% | 0~100% |
| 开屏广告 | ✅ | 展示时长/可跳过时间可配 |
| 播放中广告 | ✅ | 间隔/时长/可跳过时间可配 |
| 图片来源 | Bing 今日 | 5 种模式可选 |
